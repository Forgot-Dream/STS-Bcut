# STS-Bcut

使用必剪API，语音转字幕。

- 支持输入视频文件自动提取音频（ffmpeg）
- 支持拖动添加文件
- 支持多视频/语音文件的一次性转录

运行需要安装.NET 6 Runtime和ffmpeg
